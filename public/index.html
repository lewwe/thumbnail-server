Layers: Buttons 1-4 on the left switch between four saved preview/OSC setups.

Keys: Arrow keys move selection, Home/End jump start/end, PageUp/PageDown jump faster, Space sends OSC, B sends /none.

Colors: Green frame = current keyboard selection, Red frame = last preview that sent OSC.

Path mode: Enable ASCII-safe path for OSC strings to replace diacritics/non-ASCII characters.
